Font Sizer DX: Information

The current document contains the minimum for code setup, i.e., script tags for the two JavaScript files, the sizer links, and an init function call. For most uses you will want to include calls to setDefaults and set methods to tell the code what portions of your document you want affected (or excluded) when users click the links.

Using the set methods you can instruct the code to:

  • allow all text to be resized, regardless of styles sheets or font tags, or
  • allow text resize in only a few specific portions of a document's layout, or
  • exclude specific portions of a document from text resize.

View the source code of this document and scroll to the bottom to see example uses of these methods in place, but commented out. More information about the code, instructions for setup and additional examples are available online. Assistance in implementing the code is available to licensed users.

The code is set up with a few defaults in the dw_sizerdx.js file: 

    sizeUnit:    'px',
    defaultSize: 14,
    maxSize:     24,
    minSize:     10,

You can change these settings there if you like but more often they will be changed using the setDefaults method, for example:

// setDefaults arguments: size unit, default size, minimum, maximum
// optional array of elements or selectors to apply these defaults to
dw_fontSizerDX.setDefaults('px', 12, 9, 32, ['table#main'] );
